# React Commit 阶段完整实现：Mutation Placement 操作

## 第一部分：顶层鸟瞰

### 1. 背景问题

在前面的章节中，我们已经完成了 Render 阶段（`beginWork` 和 `completeWork`），生成了带有 `flags` 标记的 Fiber 树。但这些标记只是"计划"，真正的 DOM 操作需要在 Commit 阶段的 Mutation 子阶段执行。

本章要解决的核心问题：
- **如何遍历 Fiber 树找到所有需要操作的节点？**（利用 `subtreeFlags` 和 `MutationMask` 快速定位）
- **如何将标记为 `Placement` 的节点插入到正确的 DOM 位置？**（`commitPlacement`）
- **如何找到父 DOM 容器？**（`getHostParent`）
- **如何处理复杂的 Fiber 树结构（HostComponent / HostText / FunctionComponent）？**（递归插入）

### 2. 宏观流程图

```
commitRoot(root)
    ↓
commitMutationEffects(finishedWork)  ← 入口
    ↓
遍历 Fiber 树（DFS 向下遍历）
    ↓
    while (nextEffect !== null)
    ↓
    检查 subtreeFlags & MutationMask  ← 子树是否有副作用？
    ↓  YES
    递归处理 child
    ↓
commitMutationEffectsOnFiber(nextEffect)  ← 处理当前节点
    ↓
    检查 flags & Placement  ← 是否需要插入？
    ↓  YES
commitPlacement(finishedWork)  ← 执行 DOM 插入
    ↓
    1. getHostParent(finishedWork)  ← 找父 DOM
    ↓
    2. appendPlacementNodeIntoContainer(finishedWork, hostParent)  ← 递归插入
       ↓
       if (HostComponent || HostText)
           appendChildToContainer(stateNode, hostParent)  ← 真正的 DOM 操作
       else
           递归处理 child
    ↓
    3. finishedWork.flags &= ~Placement  ← 清除 Placement 标记
    ↓
遍历 sibling（DFS 横向遍历）
    ↓
遍历 return（向上回溯）
```

### 3. 关键技术栈

| 技术点 | 说明 |
|--------|------|
| **DFS 遍历策略** | 使用 child → sibling → return 的深度优先遍历，确保按正确顺序处理节点 |
| **subtreeFlags 优化** | 通过 `subtreeFlags & MutationMask` 快速判断子树是否需要处理，避免无效遍历 |
| **位运算清除标记** | 使用 `flags &= ~Placement` 清除已处理的副作用标记 |
| **递归插入策略** | `appendPlacementNodeIntoContainer` 递归处理不同类型节点（HostComponent/HostText/FunctionComponent） |
| **类型断言** | `parent.stateNode as Container` / `as FiberRootNode` 处理不同的父节点类型 |

## 第二部分：实战步骤

### 步骤 1：实现 commitWork 完整逻辑

**文件路径**：`packages/react-reconciler/src/commitWork.ts`

**修改内容**：
1. 导入依赖：`MutationMask`, `NoFlags`, `HostComponent`, `HostRoot`, `HostText`, `Container`, `appendChildToContainer`
2. 实现 `commitMutationEffects` 函数（DFS 遍历入口）
3. 实现 `commitMutationEffectsOnFiber` 函数（处理单个 Fiber 节点）
4. 实现 `commitPlacement` 函数（处理 Placement 插入）
5. 实现 `getHostParent` 函数（查找父 DOM）
6. 实现 `appendPlacementNodeIntoContainer` 函数（递归插入 DOM）

**代码实现**（新建文件，完整代码）：

```typescript
import { MutationMask, NoFlags } from './fiberFlags';
import { FiberNode, FiberRootNode } from './fiber';
import { HostComponent, HostRoot, HostText } from './workTags';
import { Container, appendChildToContainer } from './hostConfig';

let nextEffect: FiberNode | null = null;

export const commitMutationEffects = (finishedWork: FiberNode) => {
  nextEffect = finishedWork;

  while (nextEffect !== null) {
    // 向下遍历
    const child: FiberNode | null = nextEffect.child;

    if (
      (nextEffect.subtreeFlags & MutationMask) !== NoFlags &&
      child !== null
    ) {
      nextEffect = child;
    } else {
      // 向上遍历 DFS
      up: while (nextEffect !== null) {
        commitMutationEffectsOnFiber(nextEffect);
        const sibling: FiberNode | null = nextEffect.sibling;

        if (sibling !== null) {
          nextEffect = sibling;
          break up;
        }

        nextEffect = nextEffect.return;
      }
    }
  }
};

const commitMutationEffectsOnFiber = (finishedWork: FiberNode) => {
  const flags = finishedWork.flags;

  if ((flags & Placement) !== NoFlags) {
    commitPlacement(finishedWork);
    finishedWork.flags &= ~Placement;
  }

  // flags Update
  // flags ChildDeletion
};

const commitPlacement = (finishedWork: FiberNode) => {
  if (__DEV__) {
    console.warn('执行Placement操作', finishedWork);
  }

  // parent DOM
  const hostParent = getHostParent(finishedWork);
  // finishedWork ~~ DOM append parent DOM
  appendPlacementNodeIntoContainer(finishedWork, hostParent);
};

function getHostParent(fiber: FiberNode): Container {
  let parent = fiber.return;

  while (parent) {
    const parentTag = parent.tag;
    // HostComponent HostRoot
    if (parentTag === HostComponent) {
      return parent.stateNode as Container;
    }

    if (parentTag === HostRoot) {
      return (parent.stateNode as FiberRootNode).container;
    }

    parent = parent.return;
  }

  if (__DEV__) {
    console.warn('未找到host parent');
  }
}

function appendPlacementNodeIntoContainer(
  finishedWork: FiberNode,
  hostParent: Container
) {
  // fiber host
  if (finishedWork.tag === HostComponent || finishedWork.tag === HostText) {
    appendChildToContainer(finishedWork.stateNode, hostParent);
    return;
  }

  const child = finishedWork.child;
  if (child !== null) {
    appendPlacementNodeIntoContainer(child, hostParent);
    let sibling = child.sibling;

    while (sibling !== null) {
      appendPlacementNodeIntoContainer(sibling, hostParent);
      sibling = sibling.sibling;
    }
  }
}
```
