# 1.9-测试Demo实现

## 第一部分：顶层鸟瞰

### 1. 背景问题

在完成了 React 和 ReactDOM 的核心功能（Fiber 架构、beginWork、completeWork、Commit 阶段、Placement 操作等）后，需要验证这些功能是否正常工作。这需要：

1. **将打包后的包链接到全局**：让外部项目能够使用本地开发的 react 和 react-dom
2. **创建测试项目**：使用链接的包创建一个简单的 React 应用
3. **验证核心流程**：确认整个渲染流程（从 JSX 到 DOM 插入）是否正常执行

### 2. 宏观图

```
┌─────────────────────────────────────────────────────────┐
│              React 项目测试验证流程                        │
└─────────────────────────────────────────────────────────┘
                           │
        ┌──────────────────┼──────────────────┐
        ↓                  ↓                  ↓
   步骤1：构建          步骤2：链接         步骤3：验证
   本地包               测试项目           运行结果
        │                  │                  │
        ├─ pnpm build:dev ├─ 链接 react 包    ├─ 查看控制台
        ├─ pnpm link      ├─ 链接 react-dom   ├─ 验证 Placement
        │  --global       ├─ 导入包          ├─ 查看DOM
        ↓                  ↓                  ↓
   全局可用           demo 项目           验证 Placement
```

### 3. 关键技术栈

| 技术 | 说明 |
|------|------|
| pnpm link --global | 将本地包链接到全局 npm registry |
| pnpm link <pkg> | 将全局包链接到当前项目 |
| ReactDOM.createRoot | React 18 的根节点创建 API |
| Placement 操作 | Commit 阶段的 DOM 插入操作 |
| 调试输出 | 通过 console.log 验证执行流程 |

### 步骤 5：编写测试代码

**路径：** `src/index.js`（测试项目）

**修改内容：**
1. 导入 React 和 ReactDOM
2. 创建 JSX 结构
3. 获取根节点并调用 createRoot + render
4. 添加调试输出

```javascript
import React from "react";
import ReactDOM from "react-dom";

const jsx = (
  <div>
    hello <span>big-react</span>
  </div>
);

const root = document.querySelector("#root");

ReactDOM.createRoot(root).render(jsx);

console.log(React);
console.log(jsx);
console.log(ReactDOM);
```

### 步骤 6：修复 appendChildToContainer 参数顺序

**路径：** `packages/react-reconciler/src/commitWork.ts`

**修改内容：**
修正 `appendPlacementNodeIntoContainer` 中 `appendChildToContainer` 的参数顺序（从 `(child, parent)` 改为 `(parent, child)`）

```diff
  function appendPlacementNodeIntoContainer(
    finishedWork: FiberNode,
    hostParent: Container
  ) {
    // fiber host
    if (finishedWork.tag === HostComponent || finishedWork.tag === HostText) {
-     appendChildToContainer(finishedWork.stateNode, hostParent);
+     appendChildToContainer(hostParent, finishedWork.stateNode);
      return;
    }
    // ...其他代码...
  }
```
